# 基于 Python 的电商销售数据可视化分析

## 1. 项目背景
随着电商行业竞争日益激烈，单纯依靠经验进行运营已无法满足市场需求。本项目旨在利用 Python 对店铺实际销售数据进行清洗、整合与深度分析。通过挖掘月度销售趋势、商品品类偏好以及用户消费行为，为店铺在库存管理、促销策略制定等方面提供可视化的数据支持，从而提升市场竞争力。

## 2. 数据说明
本项目使用的数据集包含两张核心表格：
* **商品信息表**：包含商品大类（如运动上装）、商品小类（如瑜伽背心）、商品名称及货号。
* **销售数据表**：包含订单日期、订单编号、关联的商品编号及订单数量。
* **数据处理**：通过 Python 的 `pd.merge` 函数，以“商品编号”为键将两表合并，并计算了“销售金额”字段。

## 3. 分析方法与工具
* **工具栈**：Python 3.9, Pandas (数据处理), Matplotlib & Seaborn (可视化), GitHub (版本控制)。
* **核心分析维度**：
    1.  **总体趋势分析**：使用折线图监测每月销售金额的波动情况。
    2.  **品类细分分析**：筛选 Top5 畅销小类，分析其季节性销售差异。
    3.  **热力图挖掘**：利用 `Heatmap` 直观展示“商品小类 x 月份”的爆款分布。
    4.  **用户与时间行为拓展**：
        * 通过箱线图（Boxplot）分析订单客单价分布，识别核心价格带。
        * 通过柱状图分析“星期几”的销售规律，辅助营销排期。

## 4. 分析结论
通过对数据的可视化分析，得出以下关键结论：
1.  **季节性趋势**：店铺销售额在 **[7月（夏季）和10月（秋冬季）]** 达到峰值，呈现出明显的季节性波动， **[3月、5月各出现小高峰]** ，建议在旺季前 1 个月完成备货。
2.  **爆款品类**：热力图显示，**[外套]** 在夏季和冬季都表现突出，而 **[长袖]** 在冬季销量会回升。
3.  **消费水平**：用户平均客单价集中在 **[0-2000元]** 之间，且存在少量高消费异常值，说明存在高价值客户挖掘空间。
4.  **营销时间窗口**：数据显示 **[周一和周四]** 的销售额最高，建议将直播或促销活动安排在此时段。
