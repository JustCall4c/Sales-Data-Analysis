# 商品销售数据分析项目

## 1. 项目背景
本项目旨在利用 Python 对店铺销售数据进行分析，挖掘销售趋势和用户行为，为店铺的运营策略调整提供数据支持。

## 2. 数据说明
数据来源于电商平台的商品销售记录，包含商品编号、类别、销售价、订单日期等信息。

## 3. 分析方法
- 工具：Python (Pandas, Matplotlib, Seaborn)
- 方法：
    - 数据清洗与合并
    - 月度销售趋势分析（折线图）
    - 商品小类热力图分析
    - 星期维度销售分析
